Quick Setup

  1. Install OpenClaw: npm install -g openclaw@latest
  2. Run onboarding: openclaw onboard --install-daemon
  3. The gateway WebSocket runs on ws://127.0.0.1:18790
  4. Copy your API token from the config file below
